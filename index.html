CareerGo

This project helps computer science students choose the right career path using machine learning. It takes input from users about their skills and interests, then suggests suitable careers based on that data. The system was built using Python, Flask, and machine learning libraries like TensorFlow and scikit-learn.

TerraConnect

TerraConnect is a website that connects land buyers and sellers directly, without any middlemen. It shows property listings with location details using Google Maps. The platform was developed using HTML, CSS, JavaScript, PHP, and MySQL.

Personal Portfolio Website

A fully responsive personal portfolio website designed and developed using HTML, CSS, Bootstrap 5, and JavaScript. It showcases my skills, projects, resume, and contact information. The site is clean, modern, and mobile-friendly, and includes smooth layout structure and responsive grid alignment.

LifeLoop

I designed and developed a responsive appointment scheduling page for LifeLoop, a blood bank platform. The page features a dynamic date and time picker, a user-friendly form for personal details, service selection checkboxes, and a message field. Built with HTML, CSS, and JavaScript (Flatpickr), it streamlines user interaction and prepares the system for backend integration using PHP or APIs for real-time booking.
